Directories
-----------

User code
---------
Src - should contain all the user's source files
Inc - should contain all the user's header files
Docs - user documentation files

System code
-----------
StartUp - configuraton code for the board
Drivers - uVision's board level drivers (we are replacing the ones we need)

Files generated by uVision during compilation
---------------------------------------------
WM - object files
RTE - run time code
